# Lab Bookkeeper

Lab Bookkeeper — это приложение для ведения лабораторной бухгалтерии. Оно позволяет создавать и вести учёт реагентов лаборатории.

## Технологии

- Rust
- egui (графический интерфейс)

## Возможности

- Составление каталога реагентов
- Формирование статистики использования
- Хранение истории использования
- Создание кастомных шаблонов (например, заявки на приобретение)
- Хранение данных в одном файле или нескольких конфигурационных файлах для простого переноса базы данных

## Установка и запуск

### Требования

- Rust (версия 1.95.0 или новее)

### Сборка из исходников

```bash
git clone https://github.com/HexDimay/lab-bookkeeper
cd lab-bookkeeper
cargo run --release
```
